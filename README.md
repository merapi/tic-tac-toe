# tic-tac-toe
React, Redux, Socket.io

```
git clone https://github.com/merapi/tic-tac-toe.git
cd tic-tac-toe
npm install

npm start || webpack-dev-server //run dev server

npm run server //run socket server for multiplayer mode
npm run server:watch //run socket server in watch mode (editing)

npm test
npm run test:watch
```
